# Twitter Clone

[Live demo](https://twitter-clone-flawn.vercel.app/)

Built with:

- React
- React Router
- Vite
- Firebase

## Screenshot

![gif](https://github.com/alkimcaner/portfolio/blob/main/public/assets/twitter.gif)

## Project Setup

### Install dependencies

```bash
npm install
# or
yarn install
```

### Start dev server

```bash
npm run dev
# or
yarn dev
```

### Build for production

```bash
npm run build
# or
yarn build
```
