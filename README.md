# Twitter Clone

[Live demo](https://twitter-clone-alkimcaner.vercel.app/)

Built with:

- React
- React Router
- Vite
- Firebase

## Screenshot

![gif](https://github.com/alkimcaner/portfolio/blob/main/public/assets/twitter.gif)

## Project Setup

### Install dependencies

```bash
yarn install
```

### Start dev server

```bash
yarn dev
```

### Build for production

```bash
yarn build
```
